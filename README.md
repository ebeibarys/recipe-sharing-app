# Recipe Sharing App

A full-stack web application where users can create, view, like, save, and comment on recipes.

Built with Angular (frontend) and Django + DRF (backend).

## Features
- User authentication (JWT login/logout)
- Create, update, delete recipes (CRUD)
- Add ingredients and instructions
- Comment on recipes
- Like / save recipes
- Filter by difficulty (easy, medium, hard)

---

## Tech Stack
Frontend: Angular, TypeScript, HTML, CSS  
Backend: Django, DRF, SQLite  

---

## Models
- User
- Recipe
- Ingredient
- Comment
- SavedRecipe

(Relations via ForeignKey)

---

## API Examples
- GET /api/recipes/
- POST /api/recipes/
- GET /api/recipes/<id>/
- POST /api/comments/
